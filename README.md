# OpenMed
## Requirements
Software
- Python >= 3.5

Packages
- PySimpleGUI >= 4.17
- firebase >= 3.0.1

## Tutorial
**Note: To run, you must have the API Key in the OpenMed folder. You can requrest this key by emailing sam.jaros.sj@gmail.com.**
